this is the project folder for the proposed project 

Title: Realtime Phishing detection and monitoring system.


Use the requirements.txt file to install all the required python libraries

keep your html document under the name of template folder.

there are 4 python scripts "app.py", "email_fetcher.py", "phishing_detection.py", "ifdof.py"

keep your dataset to be loaded under the name of dataset.

Save the cred.env file which contains the credentials for the email account in the project folder itself.

app.py script contains the flask application module

email_fetcher.py contains the modules for extracting mail form the Gmail server.

phishing_detection.py script contains the main detection algorithm (preprocessing, labelling and classifying)

ifdof.py script contains the external evaluation program with same saved and vectorised random forest model.
